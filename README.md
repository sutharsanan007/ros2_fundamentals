# ROS 2 Fundamentals Examples

This package contains examples demonstrating fundamental ROS 2 concepts

## Description

This Package includes minimalist ROS 2 code to demonstrate important ROS 2 concepts and patterns.
- Following ROS 2 Python and C++ style guidelines
- Understanding ROS2 frameworks & getting used to the architecture of Node creation in ROS2

## Prerequisites

- ROS 2 must be installed
- Python 3
- Created a ROS 2 workspace (`ros2_ws`)
